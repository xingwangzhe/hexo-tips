# hexo-tip
A more convenient way to use tips in Hexo

[中文](README.md)

## Usage

General format:

```
:::name

content

:::
```

For example:
```
:::warning
 
This is a warning message

:::
```

Currently supported types:
+ warning: '⚠' - Warning message
+ danger: '⛔' - Danger alert
+ tip: '💡' - Tips
+ mention: '💬' - Mention
+ recommend: '👍' - Recommendation
+ note: '📝' - Notes
+ info: 'ℹ️' - Information
+ success: '✅' - Success
+ error: '❌' - Error
+ bug: '🐛' - Bug
+ todo: '📋' - Todo
+ example: '🔍' - Example
+ quote: '💭' - Quote
+ link: '🔗' - Link
+ code: '💻' - Code
+ update: '🔄' - Update
+ star: '⭐' - Star
+ time: '⌛' - Time
