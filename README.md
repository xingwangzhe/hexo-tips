# hexo-tip
更便捷地使用小贴士

[English](README_EN.md)

## 使用方法

通用格式

```
:::name

content

:::
```
比如说
```
:::warning
 
这是一个警告提示

:::

```

目前支持
+ warning: '⚠' - 警告信息
+ danger: '⛔' - 危险提示
+ tip: '💡' - 小贴士
+ mention: '💬' - 提及
+ recommend: '👍' - 推荐
+ note: '📝' - 笔记
+ info: 'ℹ️' - 信息
+ success: '✅' - 成功
+ error: '❌' - 错误
+ bug: '🐛' - 漏洞
+ todo: '📋' - 待办
+ example: '🔍' - 示例
+ quote: '💭' - 引用
+ link: '🔗' - 链接
+ code: '💻' - 代码
+ update: '🔄' - 更新
+ star: '⭐' - 重点
+ time: '⌛' - 时间
